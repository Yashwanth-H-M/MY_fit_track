# FitTrak — Premium Gym Routine 🏋️‍♂️⚡

**FitTrak** is a high-performance, mobile-first web application designed to track your weekly gym routines with a clean, premium aesthetic. Optimized for both speed and usability, it features a custom reset flow, offline support (PWA), and a sleek typography-driven design.

**Live Demo:** [https://fit-trak-five.vercel.app/](https://fit-trak-five.vercel.app/)

![FitTrak Preview](https://github.com/Yashwanth-H-M/MY_fit_track/raw/main/public/icon.png)

## ✨ Key Features

- **📱 Mobile-First Design**: Fully responsive layout optimized for handheld devices.
- **⚡ Progressive Web App (PWA)**: Install directly to your home screen for a native app feel.
- **🌐 Offline Capability**: Access your routine even without an internet connection at the gym.
- **↺ Custom Reset Flow**: A two-step "Reset All" confirmation to prevent accidental data loss.
- **📅 Weekly Split**: 7-day workout routine with dedicated "Rest & Recovery" sections.
- **💎 Premium UI**: Built with a focus on typography (Playfair Display & Inter) and smooth transitions.
- **🚀 Vercel Ready**: Optimized for seamless deployment on Vercel.

## 🛠️ Tech Stack

- **React 18** (Vite + SWC)
- **Vanilla CSS** (Custom Design System with CSS Variables)
- **Service Workers** (for PWA & Offline Support)
- **LocalStorage** (Persistent Progress)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Yashwanth-H-M/MY_fit_track.git
   ```

2. Navigate to the project directory:
   ```bash
   cd fit_trak
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create an optimized production build:
```bash
npm run build
```
To preview the build locally:
```bash
npm run preview
```

## 📱 Mobile Installation

1. Open the app link in your mobile browser (Safari on iOS or Chrome on Android).
2. Tap the "Share" button (iOS) or the three dots (Android).
3. Select **"Add to Home Screen"**.
4. Launch FitTrak from your home screen for the full-screen experience.
